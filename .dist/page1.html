Git Commands


Initialization

1-First create a repository on Github.ie MS_Assignment_1

2-Open VScode and open your folder.ie MC_ASSIGNMENT_1

3-Type Git init on Terminal to initialize

4-Type Git status to get the status of all the files present in the repository

5-Create a new file in the selected folder ie page1.html

6-Type Git add filename.extension or Git add.(to add all files) to add the files in repository



7-Type Git commit -m "Commit message" to save the current changes locally on the system.

8-Type Git push to push all the data on the online repository.





9-Git remote add origin /githublink is used to link with online repository

10- Git pull is used to get all the changes from the main online repository.
